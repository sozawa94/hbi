This directory contains some examples for input file (.in) and python script to generate the parameter file (.py).

[Non-fluid problems]
-2dp.in: a planar fault (plane strain) in a 2D elastic full space. You can also do an antiplane shear problem by setting pois=0.
-2dp_restart.in: restarting 2dp.in with a longer tmax (by running petrutbation.py before running this, you can simulate a static stress perturbation)
-2dn_rough.in: a nonplanar fault in a 2D elastic full space
-uni3dp.in: a planar fault in a 3D elastic full space
-uni3dthrust.in: a planar thrust fault (30 deg dip) in a 3D elastic half space

[Fluid problems]
-2dinjection_pressure_kpfromfile.in: constant pressure injection with non-uniform permeability from an external file
-3dinjection.in: 3D injection problem
-3dinjection_wells.in: 3D injection problem with complex injection histoty from an external file
-3dinjection_permev.in: 3D injection problem with permeability evolution by slip
-3dinjection_complexBC.in: 3D injection problem with complex boundary condition

[SCEC SEAS benchmark problems](https://strike.scec.org/cvws/seas/benchmark_descriptions.html)
-bp1.in: BP1 (vertical strike-slip fault in 2D)
-bp3dip30thrust.in: BP3 thrust fault with dip = 30 degrees
-bp5r.in: BP5 (vertical strike-slip fault in 3D) with rectangular elements 
-bp5t.in: BP5 (vertical strike-slip fault in 3D) with triangular elements 
-bp6a.in: BP6 (fluid injection) with aging law
-bp7r.in: BP7 (no initial perturbation) with rectangular elements and aging law
-bp7t.in: BP7 (no initial perturbation) with triangular elements and aging law
